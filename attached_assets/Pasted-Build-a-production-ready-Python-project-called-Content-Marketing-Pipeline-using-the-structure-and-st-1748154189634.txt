🎯 Objective

Create a CrewAI-powered workflow that:

    Researches trending topics in a given industry 

    Drafts a blog article using OpenAI

    Generates LinkedIn & X (Twitter) posts from the blog

    Prepares post schedule (CSV or text output with suggested days/times)

⚙️ Behavior

    Read seed keywords (e.g. "AI automation, B2B SaaS, productivity tools")

    Use GPT-4 to:

        Suggest 3–5 trending content topics

        Write a 300–600 word blog for the top topic

        Generate LinkedIn and Twitter posts

        Suggest publish schedule (e.g., blog Monday, LinkedIn Wednesday, X Friday)

    Save output as .md, .txt, or CSV

🔐 .env.example

OPENAI_API_KEY=your-openai-api-key

🧠 Agents and Roles

    TopicResearchAgent: Proposes trending topics from seed inputs

    BlogWriterAgent: Drafts blog posts using GPT-4

    SocialPostAgent: Generates posts for LinkedIn and X

    SchedulerAgent: Suggests post timing for max engagement
